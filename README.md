\# MELFA Kinematics Study



Repositorio de estudio para organizar y modularizar un notebook académico de cinemática de un robot industrial MELFA de 6 GDL.



\## Objetivos del proyecto



\* aprender Git y GitHub,

\* organizar código técnico,

\* separar notebooks de código reusable,

\* practicar modularización y validación básica,

\* construir una base extensible para futuras simulaciones y herramientas cinemáticas.



\## Estructura inicial



\* `docs/` → documentación del caso de estudio

\* `notebooks/` → notebook original y experimentos

\* `src/` → código reusable del proyecto
